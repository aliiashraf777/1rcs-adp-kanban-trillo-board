## 📌 Overview  
ADP Kanban Trillo Board is a modern task management web application built using React, designed to help users organize tasks efficiently using a Kanban-style workflow.  
It provides a smooth drag-and-drop inspired UI, clean structure, and responsive design for better productivity.
